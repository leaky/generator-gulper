Get started

Awesome, you're using Gulper! To get started with a totally blank template, follow the below instructions to strip out everything relating to the start page.

  • CSS ./app/assets/styles/

    a) main.scss - remove @import "startpage";

    b) delete _startpage.scss

  • Images ./app/assets/images/

    a) delete gulp.png

  • Javascript ./app/assets/scripts/

    a) app.js - remove require('./start-page');

    b) delete start-page.js

    c) delete /controllers/controller.js

  • HTML ./app/index.html

    a) delete everything inside <!-- start-page --> ... <!-- /start-page -->
